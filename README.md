Cybersecurity Awareness Chatbot

Description
This is a C# console application chatbot designed to educate users about cybersecurity topics such as phishing, password safety, and safe browsing.

features
- Voice greeting (WAV file)
- ASCII art display
- User name interaction
- Cybersecurity tips
- Input validation
- Console UI with colors and typing effect

How to Run
1. Open the project in Visual Studio
2. Ensure greeting.wav is in the project folder
3. Run the program (Ctrl + F5)

Requirements
- .NET 6.0 or later
